# 1. Platform Overview

## 1.1 What Is This Platform?

The Marketing Agent is an **autonomous AI-powered creative agency** that replicates the workflow of world-class agencies like Ogilvy and R/GA. It takes a client from brand discovery through campaign execution with minimal human intervention.

## 1.2 Core Philosophy

**AUTONOMOUS BY DEFAULT** - The AI does the heavy lifting. Humans only intervene at key approval gates:
- Strategy approval (after research)
- Concept selection (after creative ideation)
- Final approval (before publishing)

# 4. Feature Specifications

## 4.1 Onboarding System

### What It Does
Automatically researches a company's brand, market position, competitors, and target audiences by analyzing their website and public information.

### Process Flow
```
1. User enters domain
2. System crawls website (Firecrawl)
3. AI analyzes brand identity
4. AI researches market & competitors (Perplexity)
5. AI identifies target audiences
6. Results stored in Knowledge Base
7. User reviews and edits
8. User continues to Dashboard
```

### Data Collected

| Category | Fields |
|----------|--------|
| **Brand** | name, tagline, description, colors, fonts, logo, voice, values |
| **Market** | industry, position, competitors (name, strengths, weaknesses) |
| **Audiences** | segments (demographics, psychographics, pain points, goals, channels) |
| **Offerings** | products/services (features, pricing, differentiators) |
| **Context** | social handles, recent news, past campaigns, sentiment |

### Edge Cases

| Scenario | Handling |
|----------|----------|
| Domain unreachable | Error + manual entry option |
| Minimal website content | Fill with "Unknown" + flag for manual edit |
| Already onboarded | Skip to dashboard |
| Process timeout | Retry mechanism |

### What It Does

### Campaign Lifecycle


### Execution Phases
